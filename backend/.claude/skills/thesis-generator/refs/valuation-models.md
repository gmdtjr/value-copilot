# Valuation Models — DCF, Reverse DCF, Margin of Safety

---

## 1. DCF (Discounted Cash Flow)

### 기본 공식

```
내재가치 = Σ(FCFt / (1+r)^t) + TV / (1+r)^n

TV (Terminal Value) = FCFn × (1+g) / (r - g)

where:
  FCFt = t년도 잉여현금흐름
  r    = 할인율 (WACC 또는 요구수익률)
  g    = 영구 성장률 (보수적으로 GDP 성장률 수준)
  n    = 명시적 예측 기간 (보통 5~10년)
```

### 3-시나리오 접근

| 시나리오 | 성장률 가정 | 할인율 | 비고 |
|----------|------------|--------|------|
| Bear     | 낮음 (0~3%) | 높음 (12~15%) | 최악의 경우 |
| Base     | 중간 (5~10%) | 중간 (9~11%) | 가장 가능성 높은 경우 |
| Bull     | 높음 (15~20%) | 낮음 (8~9%) | 낙관적 경우 |

→ 내재가치 = Base 시나리오를 중심으로 Bear~Bull 범위로 표현

### 할인율 선택 가이드

- 버핏: 장기 국채 수익률 사용 (단순화)
- 실무: WACC = Ke×E/(D+E) + Kd×(1-t)×D/(D+E)
- 가치투자자 기본값: **10%** (기회비용 = S&P500 장기 연평균)

## 2. Reverse DCF (역방향 DCF)

> "현재 주가가 이미 어떤 가정을 내포하고 있는가?"

### 방법

1. 현재 주가와 주식수로 시가총액 계산
2. 현재 FCF에서 출발
3. 시가총액 = DCF가 되는 **내포 성장률**을 역산

### 해석

| 내포 성장률 | 판단 |
|------------|------|
| < 실제 가능 성장률 | 저평가 가능성 |
| = 실제 가능 성장률 | 적정 평가 |
| > 실제 가능 성장률 | 고평가 — 시장이 과도한 낙관 |

**Reverse DCF의 핵심**: "이 주가를 정당화하려면 앞으로 연 X% 성장해야 한다. 그게 현실적인가?"

## 3. Margin of Safety 계산

```
MoS% = (내재가치 - 현재가격) / 내재가치 × 100

매수 기준:
  MoS > 50%  : 강력 매수 (고확신 + 고불확실성 상황)
  MoS > 25%  : 매수 (일반 상황)
  MoS 0~25%  : 관망 (가격 하락 대기)
  MoS < 0%   : 고평가 (매수 불가)
```

## 4. 보조 밸류에이션 지표

### EV/EBITDA
- 산업 간 비교에 유용 (자본 구조 차이 제거)
- 성장주보다 성숙 기업에 적합

### P/FCF (주가/잉여현금흐름)
- 가치투자자가 가장 신뢰하는 배수
- P/FCF < 15: 저평가 가능성
- P/FCF > 30: 고성장 프리미엄 내포

### P/E vs. 성장률
- PEG = P/E / EPS 성장률
- PEG < 1: 저평가 시그널

## 5. 분석 데이터 부족 시 처리

공개 재무 데이터가 없거나 불충분한 경우:
1. "데이터 수집 필요: [구체적으로 어떤 데이터]" 명시
2. 프레임워크와 논리 구조만 제시
3. 절대로 숫자를 추측하거나 꾸며내지 않음
4. 분석 가능한 정성적 요소에 집중
